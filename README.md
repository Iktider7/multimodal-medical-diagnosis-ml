# Multimodal Medical Diagnosis Prediction

## Overview
This project builds a multimodal machine learning system combining:
- Medical Images
- Segmentation Masks
- Tabular Clinical Data

The model uses Vision Transformer (ViT) for image feature extraction
and an MLP network for tabular data fusion.

## Technologies
- Python
- PyTorch
- Scikit-learn
- Vision Transformer (ViT)
- Google Colab

## Features
- Image feature extraction
- Tabular data processing
- Multimodal feature fusion
- Classification model
- Cross-validation evaluation
